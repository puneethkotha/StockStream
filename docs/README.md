# StockStream Landing Page

This folder contains the product landing page for StockStream, deployed to [GitHub Pages](https://puneethkotha.github.io/StockStream/).

## Structure

```
docs/
├── index.html    # Main landing page
├── styles.css    # Styles
├── script.js     # Interactive enhancements
└── images/       # Architecture & dashboard preview images
```

## Local Preview

Open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8000 --directory docs
```

Then visit http://localhost:8000
